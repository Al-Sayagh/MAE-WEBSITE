Mae Basiratmand — Static Site
================================

Files:
- index.html — main bilingual page (FR/EN)
- styles.css — design system + layout
- script.js — interactions, language switch, accessibility, demo downloads

What works out-of-the-box:
- Smooth scrolling anchors (Services, Case Studies, Teaching, Resources, Contact)
- Language switch FR/EN
- Active nav highlight
- Calendly buttons (replace `your-calendly` with your username)
- Lead magnet & contact forms (guarded demo mode until you replace `your-email@example.com`)
- Instant offline downloads for demo files (checklist + guides)

Deploy:
- Drop the folder to Netlify / Vercel / GitHub Pages.
- Replace placeholder email/links in `index.html`.
